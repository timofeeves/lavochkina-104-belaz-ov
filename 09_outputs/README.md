# Outputs

Place issued deliverables here:

- draft ventilation concept;
- client request letters;
- calculation notes;
- final drawings/registers;
- issued PDFs/DOCX/XLSX files;
- transmittal records.

## Output Naming

Recommended pattern:

`YYYY-MM-DD_short-description_status.ext`

Examples:

- `2026-07-03_request-missing-equipment-data_draft.docx`
- `2026-07-03_ventilation-concept_draft.md`
- `2026-07-04_air-balance_calc.xlsx`
